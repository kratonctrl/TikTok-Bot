## Run Android

Clone the project 🚀

```bash
  git clone https://github.com/tahaluindo/Tiktok-ViewsBOT
```

Go to the project directory

```bash
  cd Tiktok-ViewsBOT
```

Install dependencies modules

```bash
  pip3 install -r requirements.txt
```

Start the server

```bash
  python3 main.py
```
